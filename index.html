Published

Iterator-Based Parser Combinators for Ambiguous Grammars

Param Desai, Devaansh Mann, Kyle Dewey

Computer Science Conference for CSU Undergraduates (CSCSU), 2025

Abstract

For most programming languages, context-free grammars form the basis for syntactic validity. Most literature focuses on parsing unambiguous context-free grammars, but real languages (including Swift) are sometimes ambiguous. This paper proposes a new technique for parsing ambiguous grammars: iterator-based parser combinators.

Unlike traditional parser combinators, our approach computes all parses on demand, meaning only the work specific to the parses actually used is performed. Our results show 8,812x improvement in runtime for obtaining the first valid parse on highly ambiguous grammars.

Key Contributions

  • Novel iterator-based approach serving as drop-in replacement for traditional parser combinators
  • On-demand parse computation using lazy evaluation
  • Implementation in Java with evaluation on multiple grammar types
  • Demonstrated viability for real-world ambiguous grammars like Swift
